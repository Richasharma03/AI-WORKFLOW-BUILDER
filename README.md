# ⚡ AI Workflow Builder (NextFlow)

A node-based AI workflow builder similar to n8n/Zapier, built using **Next.js, React Flow, and Google Gemini API**.

Users can visually create workflows by connecting nodes and execute real-time AI responses.

---

## 🚀 Features

- 🧩 Drag & drop node system
- 🔗 Connect nodes visually (React Flow)
- 📝 Text input node
- 🤖 LLM (AI) processing node
- ⚡ Real-time AI response using Gemini API
- ▶️ Run workflow execution engine
- 🎨 Modern UI with Tailwind CSS

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS  
- **Workflow Engine:** React Flow (@xyflow/react)  
- **Backend:** Next.js API Routes  
- **AI Integration:** Google Gemini API
